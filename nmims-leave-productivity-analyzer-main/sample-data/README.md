# Sample Data

This directory contains sample attendance data for testing the application.

## Files:
- `attendance-sample.csv` - Sample attendance data in CSV format
  
## Format:
The Excel file should have these columns:
- **Employee Name**: Name of the employee
- **Date**: Date in YYYY-MM-DD format
- **In-Time**: Clock-in time in HH:MM format (24-hour)
- **Out-Time**: Clock-out time in HH:MM format (24-hour)

## Usage:
1. Convert the CSV to Excel (.xlsx) format using Excel or Google Sheets
2. Upload the .xlsx file through the application interface
3. View analytics for each employee

## Sample Data Explanation:
- John Doe has one leave day (2024-01-10) and works Saturday half-days
- Jane Smith has one leave day (2024-01-04) and works Saturday half-days
- Both employees work Monday-Friday full days and Saturday half-days
- Sundays are off days (not included in sample)
